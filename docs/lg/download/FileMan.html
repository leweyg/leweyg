FileMan - File Splitter/Combiner

FileMan is a file fragmenation program. The point is that you could take a big file and break it into smaller files. This is useful when moving big file from one computer to another using 3 1/2 disks. And the interface is very easy to use, just select the file to fragment and it does it rest (or select the first fragmented file to recombine them into the original file)

FileMan.exe

Platform: Windows
