Evil Dice 2

Current Status: Pre-Alpha 0.7.9b


This app is currently under heavy development, this site exists only for those working on it's development. For more information, contact lewey AT leweyg.com

Stable Pre-Alpha: EvilDice2.zip (0.7.9b)


PLEASE INSTALL THE FONT: The real installer will automatically do this but for now please just install the .ttf font that is in Data directory. If you havn't installed this font IT WILL CRASH.

IF IT CRASHES AS IT STARTS: You probably need Managed DirectX, which is an official Microsoft extension to DirectX, you can get it here. Managed DirectX Installer
